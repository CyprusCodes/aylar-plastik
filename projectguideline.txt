PROJECT: Aylar Plastik Website

TYPE:
Industrial plastic manufacturing company website

MAIN GOAL:
Build a premium, modern, corporate website that reflects a real manufacturing business.
The website must NOT look like a startup SaaS product or AI-generated template.

----------------------------------------
CORE DESIGN RULES
----------------------------------------

- Clean, minimal, premium
- Industrial, not trendy
- Strong typography and spacing
- Image-driven design (real production / factory visuals)
- No visual noise, no over-design

----------------------------------------
STRICT UI RULES (CRITICAL)
----------------------------------------

- DO NOT use dashboard-style UI
- DO NOT create fake mockups or interface panels
- DO NOT use abstract placeholder blocks
- DO NOT redesign entire sections randomly
- ALWAYS preserve existing layout and improve it carefully

----------------------------------------
COLOR SYSTEM
----------------------------------------

Primary dark:
#0A1A2F (premium navy, NEVER pure black)

Accent:
Soft yellow / gold inspired by the logo
Use VERY subtly (badges, small highlights, tiny accents only)

Avoid:
- pure black backgrounds
- overly bright colors
- inconsistent color usage

----------------------------------------
HERO SECTION RULES
----------------------------------------

- Left side: text content (headline, paragraph, CTA)
- Right side: real image or image slider
- Image must be dominant and realistic
- NO UI mockups or fake panels
- NO placeholder-style visuals

----------------------------------------
SECTION STRUCTURE
----------------------------------------

Homepage sections:

1. Hero
2. Products / Solutions
3. Why Choose Us
4. About
5. Production / Process
6. Contact
7. Subscribe
8. Footer

Rules:
- Keep this order
- Do not remove sections unnecessarily
- Only refine and improve

----------------------------------------
CONTENT STYLE
----------------------------------------

- Short, clear, professional
- Industrial tone
- Avoid generic phrases like:
  "innovative solutions"
  "cutting-edge technology"

- Sound like a real manufacturer

----------------------------------------
INTERACTION RULES
----------------------------------------

- Smooth and minimal
- No flashy animations
- No heavy motion
- Subtle transitions only

----------------------------------------
DEVELOPMENT RULES
----------------------------------------

- Make small, controlled improvements
- Do NOT rewrite entire components unless explicitly asked
- Keep code clean and production-ready
- Maintain responsive design

----------------------------------------
FINAL GOAL
----------------------------------------

The website should feel like a high-end industrial manufacturer website,
NOT a template, NOT a SaaS product, and NOT AI-generated.

Before making changes:
- Analyze existing code
- Preserve what works
- Improve only what is necessary
